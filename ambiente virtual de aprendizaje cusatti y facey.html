¿Qué es Ambiente Virtual de Aprendizaje?
Sistema de Aprendizaje Virtual
El Aprendizaje Virtual consiste en utilizar la realidad en la clase, traerla de forma virtual, permitiendo al alumno hacer sus propios descubrimientos. Aunque parezca paradójico, la base del Aprendizaje Virtual es el "fracaso": equivocarse (no alcanzar una meta, cometer un error, fallar, perder...) es lo que impulsa el aprendizaje. Se trata de realizar el aprendizaje mediante la práctica, construyendo escenarios virtuales complejos, y se basa en ejercicios dirigidos o programas informáticos.

Sistema E-Learning La plataforma de e-learning, campus virtual o Learning Management System (LMS) es un espacio virtual de aprendizaje orientado a facilitar la experiencia de capacitación a distancia, tanto para empresas como para instituciones educativas. Este sistema permite la creación de “aulas virtuales”; en ellas se produce la interacción entre tutores y alumnos, y entre los mismos alumnos; como también la realización de evaluaciones, el intercambio de archivos, la participación en foros, chats, y una amplia gama de herramientas adicionales. ¿QUÉ ES E-LEARNING?
Sistema B-Learning Blended learning es el aprendizaje que combina el e-learning (encuentros asincrónicos) con encuentros presenciales (sincrónicos) tomando las ventajas de ambos tipos de aprendizajes. El b-learning es una tendencia actual tanto en el área empresarial como en el ámbito educativo. Las instituciones educativas secundarias y universitarias están adoptando este tipo de programas que permiten enseñar más allá de las aulas y atraer a los nativos digitales. ¿QUÉ ES B-LEARNING?
Sistema M-Learning El aprendizaje móvil, también conocido como m-learning, es un sistema educativo. Este sistema educativo soporta, con la ayuda de dispositivos móviles, un aceso continuo al proceso de aprendizaje. Esto puede ser en dispositivos como tu teléfono, computadora móvil o tablet. ¿QUÉ ES M-LEARNING?
